# 消息提示框 通知框对话框（Notiflix）

TIP

github 地址：Notiflix (opens new window)

文档：notify (opens new window)

下载地址：download (opens new window)

# 引用

1、 head引入css文件

<link rel="stylesheet" href="css/notiflix-1.3.0.min.css">

2、body引入js文件

<script src="assets/js/notiflix-1.3.0.min.js"></script>

# 使用

Notiflix.Notify.success('Sol lucet omnibus');

Notiflix.Notify.failure('Qui timide rogat docet negare');

Notiflix.Notify.warning('Memento te hominem esse');

Notiflix.Notify.info('Cogito ergo sum');
